Hytale está diseñado para soportar múltiples métodos de entrada para adaptarse a diferentes plataformas y estilos de juego.

## Sistema de Interacción

Hytale cuenta con un sólido sistema de interacción que permite a los plugins personalizar cómo se interpretan los atajos de teclado y qué acciones activan. El sistema está basado en datos, permitiendo a los Modders crear interacciones complejas con los recursos.

Modders pueden definir atajos de teclado personalizados para varias acciones, como usar habilidades o interactuar con objetos en el mundo. El sistema de interacción está diseñado para ser potente, aunque puede requerir cierto esfuerzo para gestionar la complejidad de los recursos involucrados.

> Puedes definir tus propias interacciones para que los atajos de teclado hagan lo que desees. El sistema de interacciones es muy robusto, pero aún necesita trabajo en su usabilidad, ya que los recursos pueden volverse complejos y difíciles de manejar.
