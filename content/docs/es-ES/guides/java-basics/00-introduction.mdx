¡Bienvenido a tu viaje en la programación de Java! Esta guía está diseñada para introducirte en los fundamentos básicos de Java.

## ¿Qué es Java?

Java es un lenguaje de programación potente y orientado a objetos creado en 1995. Es uno de los más populares en el mundo y se utiliza para todo, desde aplicaciones móviles hasta programas empresariales. Y lo más importante para nosotros, es el lenguaje que utiliza Hytale para sus servidores y su API de *modding*.

## Qué vas a aprender

Esta guía está estructurada para guiarte paso a paso por:

1. La sintaxis y lo básico de Java
2. El flujo de control
3. La programación orientada a objetos

Después nos centraremos únicamente en conceptos específicos para Hytale. (por determinar)

## Configuración del entorno

Antes de meternos en la programación, necesitarás instalar elementos esenciales:

### Kit de desarrollo de Java (JDK)

Descárgate e instala el JDK versión 25 o superior desde [la página oficial de Oracle](https://www.oracle.com/java/technologies/javase-jdk25-downloads.html) o usa una alternativa de código abierto como [AdoptOpenJDK](https://adoptopenjdk.net/).

### Entorno de Desarrollo Integrado (IDE)

Un IDE es un programa que proporciona a los programadores herramientas completas para el desarrollo de software. Entre las opciones populares para Java se incluyen:

* [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* [Eclipse](https://www.eclipse.org/)

## Cómo leer esta guía

1. Lee cada artículo en orden: los conceptos se basan unos en otros.
2. Escribe los ejemplos, no copies y pegues, te ayudará a aprender.
3. Experimenta, modifica los ejemplos para ver lo que sucede.
4. Práctica, en cada artículo incluye ejercicios.
5. Sé paciente, programar es una habilidad que toma tiempo desarrollar.

## Actitud para el *modding*

El *modding* es la resolución creativa de problemas. Harás lo siguiente:

* Añadirás nuevos bloques, objetos y criaturas a Hytale.
* Cambiarás las mecánicas del juego para crear experiencias únicas.
* Construirás sistemas que interactuaran con el mundo del juego.
* Compartirás tus creaciones con la comunidad.
