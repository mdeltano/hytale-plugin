# Hyperions

<Callout>
  Questo è un progetto fatto dalla comunità dall'utente
  [MaeMachineBroke](https://github.com/maemachinebroke).
</Callout>

**Hyperions** è una libreria API per Hytale progettata per fornire ad altre mod un'API per memorizzare, trasmettere e ricevere energia. La libreria utilizza Transfer API realizzata dal membro della comunità shailist {/* TODO link */}, ed è basato sull'API [Energy](https://github.com/TechReborn/Energy) del team TechReborn.

L'unità standard di energia utilizzata da Hyperions è chiamata Hyperion (`Hy` in breve), e un Hyperion è approssimativamente uguale a {/* TODO */}. L'unità standard del tasso di trasferimento è l'Hyperion per tick (`Hy/t`), o in alternativa, dividendo per 30, si puo' ottenere Hyperion al secondo (`Hy/s`).
