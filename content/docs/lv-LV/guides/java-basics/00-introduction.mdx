Sveicināts ceļojumā uz Java programmēšanu! Šī rokasgrāmata ir izstrādāta, lai iepazīstinātu jūs ar Java pamatiem.

## Kas ir Java?

Java ir spēcīga, objektorientēta programmēšanas valoda, kas izveidota 1995. gadā. Tā ir viena no populārākajām programmēšanas valodām pasaulē, un to izmanto visam, sākot no mobilajām lietotnēm līdz uzņēmuma programmatūrai. Mums vissvarīgākais ir tas, ka tā ir valoda, ko Hytale izmanto saviem serveriem un modifikācijas API.

## Kas Tiks Apgūts

Šī rokasgrāmata ir strukturēta tā, lai jūs soli pa solim iepazīstinātu:

1. Javas Sintaksi un Pamatus
2. Plūsmas Kontroli
3. Objektorientētu Programmēšanu

Un pēc tam fokusēsimies tikai uz Hytale specifiskiem konceptiem. (tbd)

## Jūsu Vides Uzstādīšana

Pirms iedziļināmies programmēšanā, būs nepieciešami daži būtiski rīki, kas jāinstalē:

### Java Development Kit (JDK)

Lejuplādē un instalē JDK 25 vai jaunāku no [oficiālās Oracle saites](https://www.oracle.com/java/technologies/javase-jdk25-downloads.html) vai izmanto atvērta avota alternatīvu [AdoptOpenJDK](https://adoptopenjdk.net/).

### Integrētā Izstrādes Vide (IDE)

IDE ir lietojumprogramma, kas programmētājiem nodrošina visaptverošas iespējas programmatūras izstrādei. Populārākās Java izvēles ir šādas:

* [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* [Eclipse](https://www.eclipse.org/)

## Kā izmantot šo Rokasgrāmatu

1. Izlasiet katru rakstu secībā - Jēdzieni balstās viens uz otru
2. Ierakstiet piemērus - Nekopējiet un neielīmējiet; rakstīšana palīdz mācīties
3. Eksperimentē - Modificējiet piemērus, lai redzētu, kas notiek
4. Prakse - Katrs raksts ietver vingrinājumus
5. Esiet pacietīgs - programmēšana ir prasme, kuras attīstīšana prasa laiku

## Modificētāja Domu Gājiens

Modding ir radoša problēmu risināšana. Jūs:

* Pievienosiet jaunus blokus, priekšmetus un radības iekš Hytale
* Mainīsiet spēles mehānismus, lai radītu unikālas pieredzes
* Būvēsiet sistēmas, kas saskarās ar spēles pasauli
* Dalīsieties ar rezultātiem kopienā
