# Hyperions

<Callout>
  Šis ir kopienas veidots projekts, ko uzsāka lietotājs [MaeMachineBroke](https://github.com/maemachinebroke).
</Callout>

**Hyperions** ir Hytale API bibliotēka, kas paredzēta, lai nodrošinātu citus mod ar API enerģijas uzglabāšanai, pārraidei un saņemšanai. The library uses the Transfer API by community member shailist {/* TODO link */}, and is based off of the [Energy](https://github.com/TechReborn/Energy) API by the TechReborn team.
