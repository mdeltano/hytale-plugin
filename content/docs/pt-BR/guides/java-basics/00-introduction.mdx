Bem-vindo à sua jornada em Programação Java! Este guia foi projetado para lhe apresentar ao básico do java.

## O que é Java?

O Java é uma linguagem de programação poderosa e orientada a objetos, criada em 1995. É uma das linguagens de programação mais populares do mundo e é usada para tudo, desde aplicativos móveis até software empresarial. E o mais importante para nós, é a linguagem que Hytale usa para seus servidores e API de modificação.

## O que você vai aprender

Este guia foi estruturado para ajudá-lo passo a passo:

1. Sintaxe e Basicos de Java
2. Controle de fluxo
3. Programação Orientada a Objetos

Em Seguida iremos focar apenas nos conceitos especificos ao hytale. (A fazer)

## Configurando seu ambiente

Antes de mergulharmos na programação, você vai precisar de algumas ferramentas excrescenciais, você precisa instalar:

### Kit de Desenvolvimento Java (JDK)

Baixe e instale JDK 25 ou mais do [site oficial da Oracle](https://www.oracle.com/java/technologies/javase-jdk25-downloads.html) ou use uma alternativa de código aberto como a [AdoptOpenJDK](https://adoptopenjdk.net/).

### Ambientes de Desenvolvimento Integrado (IDE)

Uma IDE é um aplicativo de software que oferece recursos facilitadores para programadores durante o desenvolvimento de software. Escolhas populares para Java incluem:

* [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* [Eclipse](https://www.eclipse.org/)

## A mentalidade de modding

Modding é uma resolução criativa de problemas. Você irá:

* Adicionar novos blocos, items e criaturas para o Hytale
* Mudar mecânicas de jogo para criar experiencias únicas
* Construir sistemas que interagem com o mundo do jogo
* Compartilhar suas criações com a comunidade
