# Hyperions

<Callout>
  Este é um projeto criado pela comunidade e iniciado por [MaeMachineBroke](https://github.com/maemachinebroke).
</Callout>

**Hyperions** é uma biblioteca de API para Hytale projetada para fornecer a outros mods uma API para armazenar, transmitir e receber energia. A biblioteca usa a API Transfer do membro da comunidade shailist {/* TODO link */}, e é baseada na API [Energy](https://github.com/TechReborn/Energy) da equipe TechReborn.

A unidade padrão de energia usada por Hyperions é chamada de Hyperion (`Hy` por curto), e um Hyperion é aproximadamente igual a {/* TODO */}. A unidade padrão da taxa de transferência é o Hyperion por tick (`Hy/t`), ou como alternativa, dividir em 30 para obter Hyperions por segundo (`Hy/s`).
