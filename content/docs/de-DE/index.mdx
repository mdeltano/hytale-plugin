---
title: Einleitung
description: Notwendiges Wissen zum Modden von Hytale.
---

Hytale ist ein Voxel-basiertes Sandbox-RPG-Spiel von **Hypixel Studios**, wie auf der [Hytale Website](https://hytale.com/) beschrieben:

> Begeben Sie sich auf ein Abenteuer, das sowohl zum Gestalten als auch zum Spielen einlädt. Hytale verbindet die Freiheit eines Sandbox-Spiels mit der Dynamik eines RPG: Erkunden Sie eine prozedural generierte Welt voller Dungeons, Geheimnisse und einer Vielzahl von Kreaturen und gestalten Sie sie dann Block für Block.

Hytale wurde von Grund auf unter Berücksichtigung von Modding und Benutzergenerierten Inhalten entwickelt.
Das Ziel von **Hytale Modding** ist es, eine hervorragende Ressource bereitzustellen, so, dass jeder
in der Lage ist, Hytale für sich selbst zu modden, unabhängig vom Skill-Level oder Spezialisierung.

Es ist wichtig zu beachten, dass Hytale Modding nicht nur auf die Programmierung von Plugins beschränkt ist.
Programmieren, visuelle Skripte schreiben, Texturieren, 3D-Modellieren, Assets modifizieren, usw. gehört alles zum Modden.

<u>In Hytale kann jeder ein Modder sein.</u>
